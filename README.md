# The Dream Journey Archive

The Dream Journey Archive is a cultural listening and documentation space created to preserve and celebrate musical heritage with care, respect, and continuity.

This project is my gratitude and tribute to **The Dream Journey**. Their work helped me through so much, and this archive is my way of honoring what they and their team have curated for the world.

It is built as a long-term home for discovery, reflection, and appreciation of performances, traditions, and memory.

## Why This Archive Exists

- To keep meaningful musical work accessible in one place.
- To support deeper listening beyond algorithmic feeds.
- To preserve cultural continuity for future generations.

## Focus

- Pakistani classical traditions
- Sufi repertoire
- Qawwali collections
- Regional musical heritage

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Supabase
- Vercel
- Youtube Data API
